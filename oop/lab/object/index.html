Task

Build a «tictactoe» game.

The game board should be created by using JPanel() container with layout = Gridlayout(3,3). Cells of a game board should be implemented by the objects of the JButton type.

A variable “symbol” should keep a symbol of current player (“X” or “O”). Clicking on the button with the empty text field should lead to write the symbol of the current player to the button’s text field and to change this symbol. The game should be over when there are 3 equals symbol in a horizontal, vertical or diagonal row or there is not an empty button.

1. Kotlin basic

Theory

Slides

Task

The purpose of the task is to introduce to the Kotlin language. The object of the task is a program for searching anagrams. Anagrams are words with the same letters, but arranged in varied order, for example, «restful» — «fluster». The program should include several functions, that prepares a list of the words for fast searching. The list of the words can be obtained here (file «20k.txt»).

Create and test the following functions:

  1. Create the function 
      fun charSort(word: String): String
    that returns a string with the letters from the argument in ascending order. You can use any algorithm of the sorting. Hint: use CharArray type.
  2. Let define 
      class WordWithIndex(val word:String, val index:Int)
    Create a function 
      fun wordArraySort(array: Array<WordWithIndex>)
    that sorts array in descending order by word property.
  3. Create a function 
      fun getAnagrams(words: Array<String>) :ArrayList<ArrayList<String>>
    that returns the list of the anagram lists, for example {{evil, live, vile, veil},{lime, mile}, …}. This function makes an array of WordWithIndex from the source array, then uses function charSort to make new array WordWithIndex with the sorted letters in each word. Next, the function wordArraySort is used to sort last array. In the sorted array, anagrams must be arranged in order and are found in one loop over the array.
  4. In IDEA project, make “resource” directory, mark it as “Resources Root” and put the file with the word list into it. Create a function 
      main ()
    that finds and prints all anagrams from the source file. To read the word list you can use the following code: 
     
    val words = WordWithIndex::class.java.getResource("20k.txt")
        .readText()
        .split("\n")
        .filter { it.length>10 }
        .toTypedArray()

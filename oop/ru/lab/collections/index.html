4 Коллекции

Теория

Пример проекта

Задание

  • Улучшите класс Board добавив в него коллекцию множеств соседей для каждой ячейки доски. Эта коллекция создается во время инициализации объекта класса с помощью функции getNeighbors. Обновите другие классы проекта для работы с этой функцией и выполните все тесты.
  • Улучшите класс Dict добавив функцию getWords которая возвращает списков слов в которые входит переданная функции подстрока. Добавьте кэширование результатов функции в карту hash. Протестируйте эту функцию.
  • Улучшите класс Thinker добавив функци getBestMove которая находит лучший ход сравнивая возможные ходы с возможными ответными ходами. Протестируйте эту функцию.
  • Реализуйте игру `Balda с графическим интерфейсом (факультативно).
